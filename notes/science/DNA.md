# DNA

DNA — deoxyribonucleic acid — is the molecule that stores the genetic instructions of every living organism, from bacteria in the [[Sahara Desert]] to the towering trees of the [[Amazon Rainforest]]. Its elegant double-helix structure was revealed in 1953 by Watson, Crick, Franklin, and Wilkins.

The code is written in just four chemical letters — A, T, C, and G — whose sequence spells out the proteins that build and run cells. The same molecule enables [[Photosynthesis]] machinery to be inherited by each new generation of plants.

The Human Genome Project, completed in 2003, read all three billion letters of human DNA. Since then, CRISPR gene editing has made it possible to rewrite that code with startling precision, raising profound ethical questions.

Modern biology increasingly leans on [[Artificial Intelligence]]: systems like AlphaFold predict how the proteins encoded by DNA fold into their functional shapes, a puzzle that stumped scientists for fifty years.
